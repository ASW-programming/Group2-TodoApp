# Welcome to Group 2 Todo-app.

1. git clone the project.
2. Run `npm install` in root directory to install **Concurrently**
3. `CD frontend` and run `npm install` to install **React + Vite**
4. `CD ..` to go back to root
5. `CD backend` and run `npm install` to install **Firebase + Express + Cors + Nodemon**
6. `npm run dev` in root directory to start Frontend + Backend
